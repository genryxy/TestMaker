# Android Mobile Application for Tests Drafting and Passing (TestMaker)
 Opportunities:
1) Create tests with one/many answers or own answer, attach images to the questions and select an image as the test logo
2) Pass tests
3) Select questions from a specific category (FireBase) 
4) Select questions from a specific test (FireBase)
5) Shuffle questions
6) Select question from a specific category (local DB)
7) Review your answers to all the passed tests (with selected answers)
8) Review total result of other users
